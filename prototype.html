V2

We chose to developed a more streamlined and less rigid desgin. We separated each major component (Control Unit and Power Source, Sensors) into different parts in order to create a modular product, thus extending it's target audience:

This design separates the sensor components with a plug-and-play idea in mind. This way, the consumer can choose the sensors he requies with the capabilty of expansion.
The sensors that would be available with this product will record:

  • Soil Moisture
  • Air Temperature and Humidity
  • UV Intesity
  • Position of the sun
  • Sunlight Intesity
  • Motion via Infrared (Replacig the UV sensor)
  • Motion via Ultrasonic Signals (the device can act as a tripwire)
The modules draw power from a the main unit, which can be recharged via USB. An extra module that holds a rechargeable power source is implemented. This one can be recharged with a wireless charger.
